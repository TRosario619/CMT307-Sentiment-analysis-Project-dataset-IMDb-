*************************IMDb Movie Review******************************

**Overview 

This model predicts the polarity of the Movies reviews from IMDb Dataset.

**Dataset 

The core dataset contains 25,000 reviews split into train, development
and test sets. The overall distribution of labels is roughly balanced.

**Requirements:
- Anaconda installed with Python 3.6+

**Instructions to run code in Linux/Windows:

1. Unzip Zip file to specified folder
2. Open Python file with Jupyter Notebook
3. Change url path of the dataset to your local path(There is a variable url in the second cell of code). 
	Note * Keep datasets in folder structure for best results:
		IMDb
			-train
			-dev
			-test
		* keep "stopwords-en.txt" file should be in IMDb folder
4. Run File
